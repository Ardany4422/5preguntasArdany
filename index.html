CUESTIONARIO

Pregunta 1:

¿Cuál de las siguientes opciones describe mejor CSS Grid?

a Un framework para el diseño web responsivo.

b Una herramienta para crear estilos visuales en una página web.

c Una especificación de CSS que permite crear diseños de rejilla bidimensionales.

d Una biblioteca de componentes predefinidos para el diseño web.

Pregunta 2:

¿Cuál de las siguientes afirmaciones es cierta sobre CSS Grid?

a Solo es compatible con navegadores modernos.

b Es necesario utilizar un framework como Bootstrap para utilizar CSS Grid.

c Solo se utiliza para crear diseños de una sola columna.

d Permite establecer reglas de posicionamiento y alineación de elementos en una rejilla.

Pregunta 3:

¿Cuál de las siguientes opciones describe mejor el framework Bootstrap?

a Una especificación de CSS que permite crear diseños de rejilla bidimensionales.

b Una biblioteca de componentes predefinidos para el diseño web.

c Un lenguaje de programación para el desarrollo de aplicaciones web.

d Un editor visual para crear sitios web sin escribir código.

Pregunta 4:

¿Cuál de las siguientes afirmaciones es cierta sobre el uso de CSS Grid y Bootstrap juntos?

a CSS Grid reemplaza completamente a Bootstrap en la creación de diseños web.

b No es posible combinar CSS Grid y Bootstrap en un mismo proyecto.

c Bootstrap utiliza CSS Grid como su sistema de rejilla predeterminado.

d CSS Grid y Bootstrap

Pregunta 5:

¿Cuál de las siguientes afirmaciones es cierta sobre el modelo de caja en CSS?

a El modelo de caja se refiere al sistema de posicionamiento y alineación de elementos en una página web.

b El modelo de caja se utiliza para crear diseños de rejilla bidimensionales en CSS.

c El modelo de caja es una especificación de CSS que define la estructura y el estilo de los elementos rectangulares en una página web.

d El modelo de caja se basa en la propiedad display de CSS y determina cómo se representan los elementos en el navegador.
